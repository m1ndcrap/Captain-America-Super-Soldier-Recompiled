# Captain America Super Soldier Recompiled
Recompilation of the Xbox 360 version of Captain America Super Soldier with ReXGlue

> NOTES:
> This uses ReXGlue v0.4.
> It boots to title screen but crashes when loading into story mode due to missing addresses in .toml (this is being worked on).
> 
Recompiled using the ReXGlue SDK:
https://github.com/rexglue/rexglue-sdk

# Build
Follow the instructions here [ReXGlue](https://github.com/rexglue/rexglue-sdk/wiki/Guide-%E2%80%90-Getting-Started)
